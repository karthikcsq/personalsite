## Veritas (Catapult 2026 - Winner: Best Proof-of-Human Application)
Veritas is a clinical research platform that combines cryptographic proof-of-personhood with ML-powered response quality scoring to give researchers clean, trustworthy data. The team worked with and spoke to real clinical researchers who have problems with data quality, solving a fraud problem that affects 4% of a $35 billion market. An estimated 4% of trial volunteers are "professional patients" who fabricate or exaggerate symptoms for compensation.

How it works in three layers:
- Identity: Participants verify through World ID (iris biometrics) to cryptographically prove they are a unique human. You cannot enroll twice.
- Intelligence: Every response is scored across multiple quality dimensions by an ML pipeline. Researchers see per-participant integrity scores, flagged contradictions, timing anomalies, and cross-response similarity analysis from a six-tab analytics dashboard.
- Experience: AI analyzes researcher questions before the study launches, suggesting improvements. During the survey, participants get real-time nudges if responses are vague or off-topic.

The scoring pipeline combines LLM scoring (GPT-4o-mini evaluating coherence, effort, consistency, specificity) with structured scoring (response time analysis, reverse consistency checks). A RAG pipeline using Pinecone detects copied responses across participants.

Built with: Next.js 14, React 19, Tailwind CSS, shadcn/ui, PostgreSQL via Supabase, NextAuth.js, World ID, OpenAI GPT-4o, Pinecone, Vercel
Team: Karthik Thyagarajan, Sonny Chen, Melody Yang, Alex Li
GitHub: https://github.com/karthikcsq/veritas
Devpost: https://devpost.com/software/veritas-81fzu6
Demo video: https://youtu.be/aXmBme6b2-E
Live: https://veritas-kasm.vercel.app

## google-tools-mcp (Apr 2025)
An open-source MCP server that unifies all of Google Workspace behind a single server and OAuth flow. Karthik built this because he was frustrated with the fragmented state of Google integrations for AI agents. Most existing solutions required 4-5 separate MCP servers, each with their own auth flow, config, and quirks.

Key features:
- 169 tools covering Drive, Docs, Sheets, Slides, Gmail, Calendar, and Forms
- Single OAuth flow. One setup command: npx google-tools-mcp setup
- Auto re-auth: handles token expiration and scope changes seamlessly
- Reads PDFs and Word docs directly from Drive without conversion
- Read-before-edit guards: prevents blind overwrites, lists all matches when editing ambiguous content
- Can orchestrate across the entire workspace: read a PDF from Drive, summarize it into a Doc, pull data from a Sheet into a Slides deck, and email it to a team
- No telemetry, multi-account profiles, service account support for team automation
- MIT licensed, published on npm
- GitHub: https://github.com/karthikcsq/google-tools-mcp
- npm: https://www.npmjs.com/package/google-tools-mcp

## Repple (Ongoing)
Repple is a competitive fitness app for iOS with 200+ active users. Karthik is a Founding Engineer (title on LinkedIn as of Nov 2025). The app helps make fitness more fun and consistent. Users earn points for workouts, create streaks, compete against friends in weekly matchups, and climb leaderboards.

Key technical details:
- Built the entire backend using FastAPI and PostgreSQL with Supabase
- Synchronized multiplayer functionality so users can compete in real-time
- Custom ELO matchmaking algorithm that pairs users of similar fitness levels
- Automated cron-based point aggregation system
- Generative AI workout plans that adapt based on user performance metrics
- Built with Python (FastAPI), Supabase, PostgreSQL, OpenAI API, Pydantic, TypeScript, and Anthropic Claude
- Co-founded with Rishab Chakravarty, Arjun Chadha, and Ayush Guhan

Repple was born from a real motivation gap in exercise logging. The team saw that the best way to motivate consistency was by solving another problem simultaneously: a need for community and competition at the gym. Arjun Chadha reached out to Karthik out of the blue, and Karthik immediately agreed to begin working on it. Rishab Chakravarty, Arjun Chadha, and Ayush Guhan are Karthik's three cofounders.
- iOS App Store: https://apps.apple.com/us/app/repple-app/id6756220797
- LinkedIn: https://www.linkedin.com/company/reppleapp
