Q: What is google-tools-mcp?
A: google-tools-mcp is an open-source MCP server Karthik built that unifies all of Google Workspace behind a single server and OAuth flow. It provides 169 tools covering Drive, Docs, Sheets, Slides, Gmail, Calendar, and Forms. Key features include auto re-auth, reading PDFs and Word docs directly from Drive, and read-before-edit guards to prevent blind overwrites. It's MIT licensed with no telemetry, supports multi-account profiles and service account automation. Install with: npx google-tools-mcp setup.

Q: What does Karthik think about MCP and AI agents?
A: Karthik is a vocal advocate for the MCP ecosystem. He believes MCP is fundamentally "context injection" and that there is too much friction in the current experience. His vision is that MCP tool discovery should be as simple as visiting a URL, with no manual server installation or JSON configuration. He also believes businesses need to expose lightweight, standardized tool definitions so AI agents can interact with the real world natively, instead of relying on brittle browser automation or incomplete custom integrations. He built google-tools-mcp to demonstrate what a seamless MCP experience looks like.
